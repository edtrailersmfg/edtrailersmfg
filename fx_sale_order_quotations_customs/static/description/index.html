Customizaciones para Ventas

Este modulo agrega las siguientes caracteristicas:

  • Permite usar Tarifas avanzadas.
  • Permite tener Cotizaciones con calculos dinamicos.

Listas de Precios / Tarifas

Permite un calculo avanzado, agregando los siguientes valores:
  • Utilidad %
  • Tarifa Extra %
  • Tarifa Extra 2
  • Tarifa Extra 2 %



Nos permite restringir las tarifas mediante la segmentación por:
  • Vendedor
  • Equipo de Ventas



Lineas de Cotización

Permite un recalculo del precio de cada producto, tomando los siguientes parametros:
  • Tarifa Extra $
  • Tarifa Extra %
  • Descuento $
  • Descuento %




Recalculo de Cotización

Permite un recalculo del monto de la cotización, tomando los siguientes parametros:
  • Tarifa Extra $
  • Tarifa Extra %
  • Descuento $
  • Descuento %
